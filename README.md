# amazon-item-info
Little python script that will give you information about Amazon item by url link.

Information will be save in json file as:
            "title": "",
            "price": "",
            "review rating": "",
            "availability": "",
            "short info about product": "",
            "whats inside the box": "",
            "main picture url": "",
            "item url": "",
            "item type": ""
